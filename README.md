# Call Center Performance Dashboard

An interactive Excel dashboard analyzing 1,000 calls and $96,623 in revenue across a full fiscal year.

## Overview
This dashboard was built entirely in Microsoft Excel - from raw data to a fully interactive visual report. The goal was to transform messy, unstructured call center data into clear, actionable business insights.

## What's Inside
- Raw data cleaning and structuring from scratch
- Pivot tables for dynamic, multi-dimensional analysis
- Interactive slicers for rep-level and time-based filtering
- Visual storytelling through charts, KPIs, and performance matrices

## Key Metrics
| Metric | Value |
|--------|-------|
| Total Calls | 1,000 |
| Total Revenue | $96,623 |
| Average Rating | 3.9 |
| Happy Callers | 307 |
| Avg. Revenue Per Call | $96.6 |
| Total Call Duration | 89,850 mins |

## Key Insights
- R03 leads revenue efficiency at $100.8 per call
- March is the busiest month with 40 calls - 2x the monthly average
- 30–60 min calls score highest in satisfaction
- C0005 is the highest value customer at $7,747 total revenue
- Cleveland accounts for 39% of all tracked city calls

## Tools Used
- Microsoft Excel (Pivot Tables, Slicers, Conditional Formatting, Charts)
